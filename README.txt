#PLEASE READ ME#

There are two programs that are within this Folder.

1. dnd_battle_sim - the major portion of the program, carries out
character creation or loading of character, and  carries out the battle
simulator part as well.

	If you have an empty enemies folder, run the enemy generator
program to fill in the folder out with enemies if need be. To allow
the program to simulate the fight.

2. enemy_generator - program that allows the writing of enemies for the program. 
	Allows for the creation of enemies for the program!

###########################################################

Step 1. Check enemy folder and fill it up using the Enemy
generator to fill it up an populate your fight simulator.

Step 2 Run the Battle Simulator and create a character within it.

Step 3 Proceed to the fight and enjoy!
